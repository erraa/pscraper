# Pscraper

Det vet inte du

## Download
If you have git installed, clone the repository

    git clone https://github.com/erraa/pscraper

# Usage
1. Open a new screen 

    screen -S pscraper

2. Configure settings.py

    cp sample.settings.py settings.py
    
    // Setup the variables
    
    vim settings.py
   
2. Run the scraper

  python scraper.py
